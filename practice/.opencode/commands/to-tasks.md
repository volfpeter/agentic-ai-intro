description: Break an agreed plan into a structured task list
---

Break the agreed plan into a task list

Create a descriptively named subfolder in the `ai/` folder for the task list

Write each task (with the necessary information) to a descriptively named markdown file in the created folder

The filename should be prefixed with the number of the task in the list

Multiple tasks can use the same number prefix if could be executed in parallel and are of the same complexity
